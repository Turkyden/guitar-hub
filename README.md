## 🥁 非洲鼓

X => 低音

0 => 高音

~ => 切音

节奏型: [ X 0 | XX 0 ] [ R R | RL R ]

节奏型: [ X 0 | XX 00 ] [ R R | RR RL ]

节奏型: [ X 0 | XX 000 ] [ R R | RR RLR ]

节奏型: [ X ~ | XX ~00 ] [ R R | RR RLR ]
