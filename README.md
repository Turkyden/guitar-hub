## :: 非洲鼓

- X => 低音
- 0 => 高音
- ~ => 切音

Type1: [ X 0 | XX 0 ][ R R | RL R ]

Type2: [ X 0 | XX 00 ][ R R | RR RL ]

Type3: [ X 0 | XX 000 ][ R R | RR RLR ]

Type4: [ X ~ | XX ~00 ][ R R | RR RLR ]
